網頁設計簡介：
===
平常使用學校的選課系統發現沒辦法直接看到自己選好課程之課表，因此我們在期末報告的發展方向為做出登入系統及選課系統並呈現出目前已選課的課表為主。

網站架構：
===
1.登入介面→登入成功→開始選課→判斷衝堂問題→顯示課表<br/>
2.登入介面→登入失敗→登入介面<br/>
3.登入介面→註冊介面→註冊成功→登入介面<br/>
4.登入介面→註冊介面→註冊失敗→登入註冊介面<br/>
